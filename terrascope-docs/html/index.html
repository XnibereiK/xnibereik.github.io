Welcome to terrascope-ml’s documentation!¶

terrascope-ml by Terrascope.io Inc. is a repository for Satellite Image analysis of the properties and the surrounding area used in generating objective reports and personalized suggestions. This documentation includes overview of basic features and functions as well as example usage from perspective of the developer.

Note

This project is under active development. Check out the Usage section for further information, including how to install the project. End-user documentation coming soon. As of version 0.1, the interface with the user is implemented as a simple API endpoint available here.
